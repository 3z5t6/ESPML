## Project Introduction

Energy Intelligent Forecasting is a power forecasting service tool designed for the energy industry. By integrating advanced technologies such as machine learning, deep learning, automatic feature engineering, model training and incremental learning, it achieves high-precision power forecasting and optimizes energy management.

## Features

- High-precision forecasting: using deep learning models to provide accurate power forecasting.
- Automatic feature engineering: intelligently extract key features to reduce manual intervention.
- Incremental learning: supports continuous model updates to adapt to dynamically changing data.
- Visual reports: generate intuitive forecast results and analysis reports.
